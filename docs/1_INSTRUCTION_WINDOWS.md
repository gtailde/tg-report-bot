# ІНСТРУКЦІЯ 1: Запуск на Windows (Локально або на ПК)

Ця інструкція створена для початківців. Вона пояснює кожен крок: від підготовки комп'ютера до запуску бота, код якого знаходиться на GitHub.

---

## Етап 1: Підготовка програм (Робиться 1 раз)

Щоб бот запрацював, вашому комп'ютеру потрібні дві програми: **Node.js** (двигун коду) та **Git** (щоб скачати код).

1.  **Встановлення Node.js:**
    *   Зайдіть на сайт: [https://nodejs.org/](https://nodejs.org/)
    *   Скачайте версію **LTS** (вона зліва, стабільна).
    *   Встановіть як звичайну програму (натискайте Next -> Next -> Install).
    *   *Перевірка:* Натисніть `Win + R`, введіть `cmd`, натисніть Enter. У чорному вікні введіть `node -v`. Якщо показало цифри (наприклад, `v20.x.x`) — все супер.

2.  **Встановлення Git:**
    *   Зайдіть на сайт: [https://git-scm.com/download/win](https://git-scm.com/download/win)
    *   Скачайте "Click here to download" під "64-bit Git for Windows Setup".
    *   При установці просто натискайте Next (там багато вікон, нічого змінювати не треба).

3.  **Встановлення редактора коду (Рекомендовано):**
    *   VS Code ([https://code.visualstudio.com/](https://code.visualstudio.com/)) — у ньому зручно редагувати налаштування. Але можна і через Блокнот.

---

## Етап 2: Скачування бота (Клонування)

Тепер нам треба забрати код з GitHub на ваш комп'ютер.

1.  Створіть папку на Робочому столі (наприклад, з назвою `MyBot`).
2.  Зайдіть у цю папку.
3.  Натисніть **праву кнопку миші** на порожньому місці.
    *   Якщо є пункт "Open Git Bash here" або "Open in Terminal" — натисніть його.
    *   Якщо немає: у верхній стрічці провідника (де шлях папки) напишіть `cmd` і натисніть Enter.
4.  У терміналі введіть команду:
    ```bash
    git clone https://github.com/gtailde/tg-report-bot.git .
    ```
    *(Крапка в кінці важлива — вона означає "скачати в цю ж папку").*

---

## Етап 3: Встановлення бібліотек

Бот використовує готові модулі (для Телеграму, бази даних, таймерів). Їх треба завантажити.

1.  Переконайтесь, що ви в папці з ботом у терміналі.
2.  Введіть команду:
    ```bash
    npm install
    ```
3.  Зачекайте, поки з'явиться папка `node_modules`.

---

## Етап 4: Налаштування секретів (.env)

На GitHub ніколи не зберігають паролі. Тому файл налаштувань треба створити вам.

1.  У папці з ботом створіть звичайний текстовий файл.
2.  Перейменуйте його в `.env` (так, просто крапка і env, приберіть `.txt` в кінці).
    *   *Якщо Windows запитає "Ви впевнені?", тисніть "Так".*
3.  Відкрийте цей файл через Блокнот.
4.  Вставте туди цей текст і замініть дані на свої:

```ini
# Токен вашого бота (брати у @BotFather в Telegram)
BOT_TOKEN=1234567:AAA-ABBBCCC_dEeF

# Ваш особистий Telegram ID (брати у @GetMyIDBot)
# Це зробить вас головним адміном
ADMIN_ID=111222333

# (Необов'язково) ID групи для звітів Якщо залишите пустим, звіти будуть тільки в базі
# Брати у @raw_data_bot (додайте його в групу)
GROUP_ID=-100123456789
```

5.  Збережіть файл.

---

## Етап 5: Запуск!

У терміналі (у папці з ботом) введіть:
```bash
npm start
```

Якщо побачили напис **"Bot is running..."** — вітаю! Зайдіть в Телеграм і напишіть боту `/start`.

**Важливо:**
*   Бот працює, поки відкрито вікно терміналу. Якщо його закрити — бот вимкнеться.
*   База даних (`database.sqlite`) створиться автоматично після першого запуску.

---

## Часті питання:

**Q: Як отримати токен?**
A: Знайдіть в Телеграм бота **@BotFather**, напишіть `/newbot`, придумайте ім'я. Він видасть довгий ключ — це і є токен.

**Q: Як оновити бота, якщо програміст вніс зміни на GitHub?**
A: Відкрийте термінал у папці і введіть `git pull`. Потім перезапустіть бота.
